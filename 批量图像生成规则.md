# 批量图像生成规则

文件定位：图像生成规则。

用途：调用 Codex Imagen，把已经生成好的 prompt 转换为图片。本文件不负责生成 prompt，也不负责修改 prompt 内容。

生成工具：Codex Imagen

## 调用前身份读取

每次调用 Codex Imagen 生成图像前，必须先读取 `身份.md`，确认当前执行身份、固定参数和输出要求。

## 输入来源

- prompt 来源：`outputs/prompts` 中日期最近的时间文件夹。
- 风格来源：prompt 生成阶段参考 `AI人像风格测试分类.md`。
- 图像生成阶段只读取并调用已经生成好的 prompt，不重新生成、不扩写、不合并 prompt。
- 参考图来源：根据 prompt 文件名中的 `女性参考图` 或 `男性参考图`，分别使用根目录下的 `女性.jpg` 或 `男性.jpg`。

## 固定生成参数

- 模型/工具：Codex Imagen
- 质量：low
- 尺寸：1024x1024
- 输出目录：outputs/images
- 发送方式：本地保存成功后，使用 `cc connect` 发送给用户

## 每次生成图像的执行指令

每次生成图像时，统一使用 Codex Imagen，并固定使用以下参数：

```text
quality: low
size: 1024x1024
```

生成时只替换图像提示词内容，不修改质量和尺寸参数。

## 单张图像生成模板

```text
使用 Codex Imagen 生成图像。

调用前：
- 读取 身份.md
- 读取 批量图像生成规则.md

参数：
- quality: low
- size: 1024x1024

生成后：
- 保存到 outputs/images
- 使用 cc connect 发送生成好的本地图片

提示词：
{填入已经生成好的 prompt}
```

## 批量生成流程

1. 先读取 `身份.md`。
2. 再读取本文件，确认批量生成规则。
3. 扫描 `outputs/prompts` 下的时间文件夹，选择日期最近的一个文件夹。
4. 读取该最新时间文件夹里的所有 prompt 文件。
5. 根据 prompt 文件名判断参考图：
   - 文件名包含 `女性参考图` 时，使用 `女性.jpg` 作为参考图。
   - 文件名包含 `男性参考图` 时，使用 `男性.jpg` 作为参考图。
6. 每条 prompt 单独调用一次 Codex Imagen。
7. 每次调用都使用固定参数 `quality: low` 和 `size: 1024x1024`。
8. 生成结果保存到 `outputs/images`。
9. 文件名按 prompt 文件名去掉 `_prompt` 后命名，避免覆盖已有图片。
10. 每张图片保存到本地后，立即使用 `cc connect` 发送给用户。
11. 发送完成后记录本地文件名和发送状态。

## 最新 prompt 文件夹选择规则

1. 只读取 `outputs/prompts` 的直接子文件夹。
2. 优先按文件夹名称中的时间排序，选择最新的一个。
3. 如果文件夹名称无法判断时间，则按文件夹最后修改时间选择最新的一个。
4. 只读取最新文件夹内的 prompt，不读取旧批次 prompt。
5. prompt 文件建议使用 `.txt` 或 `.md` 后缀。

## 男女参考图调用规则

```text
女性参考图 prompt -> 参考图片：女性.jpg
男性参考图 prompt -> 参考图片：男性.jpg
```

每个 prompt 必须与对应参考图一起调用 Codex Imagen，保证人物主体参考一致。

## 本地保存与发送规则

1. 每张图片必须先保存到 `outputs/images`。
2. 只有确认本地文件保存成功后，才能调用 `cc connect` 发送。
3. 每张图片单独发送，发送顺序与 prompt 读取顺序保持一致。
4. 发送时使用已保存的本地图片路径，不使用临时路径或未落盘内容。
5. 如果某张图片发送失败，记录失败文件名，并继续处理后续图片。
6. 批量任务结束后，汇总本地保存文件名和 `cc connect` 发送结果。

## 命名规则

建议使用以下格式：

```text
{批次编号}_{序号}_{图像主题}.png
```

示例：

```text
20260605_001_古风人像.png
20260605_002_城市夜景.png
```

## 检查项

生成前确认：

- 是否已读取 `身份.md`。
- 是否调用的是已经生成好的 prompt。
- 是否只读取了 `outputs/prompts` 中日期最近文件夹里的 prompt。
- 是否根据文件名正确匹配 `女性.jpg` 或 `男性.jpg` 参考图。
- 是否使用 Codex Imagen。
- 是否设置 `quality: low`。
- 是否设置 `size: 1024x1024`。
- 是否为每条提示词单独生成一张图像。
- 是否将输出图片保存到 `outputs/images`。
- 是否在每张图片保存到本地后使用 `cc connect` 发送给用户。
